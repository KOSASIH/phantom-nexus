## 📦 Quick Start

### Prerequisites
- Rust 1.75+ with `wasm32-unknown-unknown` target
- Node.js 20+
- Python 3.11+ (for AI modules)

### Build
```bash
# Clone the repository
git clone https://github.com/KOSASIH/phantom-nexus.git
cd phantom-nexus

# Build all Rust crates
cargo build --release

# Run tests
cargo test --all

# Start local testnet node
cargo run --bin nexus-node -- --testnet
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
