## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/KOSASIH/phantom-nexus.git
cd phantom-nexus

# Build
cargo build --release

# Run tests
cargo test --all

# Start local node
cargo run --bin nexus-node -- --testnet

# Frontend
cd frontend && npm install && npm run dev
```
